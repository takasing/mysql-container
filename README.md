# mysql-container

If you do not want to install mysql with Homebrew, install in Vagrant.  
This repository provide you MySQL + Vagrant + Docker environment on local machine(Mac OS X).  

## Architecture
Docker(MySQL) + Docker(Volume container)  
↑  
Vagrant  
↑  
Mac OS X  

## Required
- VirtualBox
- Vagrant

## Initialize
```sh
vagrant up
```
